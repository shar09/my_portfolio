Google Books App

A MERN stack application that uses the google books API to search for books and connects to the MongoDB database to store books of interest. Developed using React for the client side and Node.js for the backend.

React Memory Game

A fun memory game developed using React. The user scores when the right image is clicked.

DevNet

Devnet is a website that lets developers with similar interests collaborate by creating events based on location and time. Developed using Google's Firebase, GitHub and Yelp API's.

Luffy's Bar

A website that allows customers to order drinks and play arcade games at the same time. Developed using Node, Express, MySQL and Handlebars.

Friend Finder

A friend finder application that calculates user compatabalities based on a survey. Developed using Node and Express.

GifTastic

A website that displays Movie and Tv Gif's based on user's choice. Developed using jQuery and GIPHY API.

Crystal Collector

A fun game to sharpen your brain. Developed using JavaScript and jQuery.

Bamazon

Liri Bot
